Aplikasi Manajemen Santri (Binduk MADIN)

Versi: 2.0.0

Deskripsi: Aplikasi ini digunakan untuk mengelola data santri, nilai, kehadiran, kepribadian, serta mencetak rapor dan leger. Dikembangkan untuk memudahkan administrasi Madrasah Diniyah.

Fitur Unggulan:

  • Data Santri (CRUD + filter kelas/tahun)
  • Data Asatid/Guru (CRUD) + Jadwal Mengajar Asatid/Guru
  • Nilai 15 mapel per semester (Tauhid, Al-Qur'an, Hadist, Akhlak, Fiqih, Bahasa Arab, Sejarah, Tajwid, Nahwu, Shorrof, B.Al-Qur'an, TBA, Al-Khot, P.Ibadah, Muhafadzah )
  • Kehadiran dan Kepribadian per semester
  • Cetak Rapor lengkap (nilai angka+huruf, total, peringkat)
  • Leger nilai dengan total & rata-rata
  • Backup/Restore JSON
  • Import/Export Excel (tambah banyak santri sekaligus)
